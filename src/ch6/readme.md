## 함수 추출하기
- 코드의 목적(강조)과 구현(반전)을 분리한다.
- '어떻게'가 아닌 '무엇을' 하는지가 드러나야한다.
- 팁(Tip)
    - 함수의 길이는 중요하지 않다.
        - 단 한줄짜리 함수를 만들어도 상관없다.
    - 성능 최적화는 지침에 따라라(하지마라)
    - 추출한 함수의 이름을 지을때 주석을 참고하면 도움이 된다.
    - *본문 코드가 이름만큼 깔끔하다면, 함수를 추출하지 않아도 된다.*
